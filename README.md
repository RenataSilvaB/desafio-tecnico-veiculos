# Gerenciamento de Veículos

Aplicação fullstack para cadastro e gerenciamento de veículos. Você consegue listar, cadastrar, editar e excluir veículos de forma simples e direta.

---

## O que foi feito

- Listagem de veículos em tabela
- Cadastro de novo veículo
- Edição de veículo existente
- Exclusão com diálogo de confirmação

---

## Tecnologias

**Frontend**
- Angular
- TypeScript
- SCSS

**Backend**
- Node.js
- Express
- JSON como banco de dados (db.json)

---

## Como rodar o projeto

### Backend

```bash
cd backend-veiculos
npm install
node server.js
```

O servidor sobe em `http://localhost:3000`

### Frontend

```bash
cd frontend-veiculos
npm install
ng serve
```

A aplicação abre em `http://localhost:4200`
